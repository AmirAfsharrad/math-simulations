Linearization: the tangent line and tangent plane

A smooth function looks linear up close. Type any function, then move the base point: drag on the plot in 1D, or use the pad and sliders in 2D. The best linear approximation is f(x) ≈ f(x0) + Df(x0)(x − x0); watch where it is good and where it breaks down.
